First off, thank you for considering contributing to MediaShare! It's people like you who make the open-source community such an amazing place to learn, inspire, and create.

## How Can I Contribute?

### Reporting Bugs

- Use the GitHub Issues tab.
- Describe the steps to reproduce the bug.
- Include screenshots if it's a UI/UX issue related to the theme.

### Suggesting Enhancements

- We are always looking to improve the "Sovereign Brown" aesthetic.
- If you have ideas for new features (e.g., video support, folder nesting), open an issue to discuss it.

### Pull Requests

1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the UI matches the design system (use Tailwind v4 variables).
4. Issue that pull request!

## Style Guide

- **TypeScript:** Use functional components and strict typing.
- **CSS:** Use the `--color-*` variables defined in `globals.css`. Do not hardcode hex values.
- **Commits:** Use descriptive messages (e.g., `feat: add download all button`).
